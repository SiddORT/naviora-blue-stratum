# File Storage Standards

IMPORTANT:

Never store files as BLOBs inside PostgreSQL.

Store only:

* File Name
* Original Name
* MIME Type
* File Size
* Relative Path
* Storage Provider

inside database.

Example:

{
"file_name": "assessment_123.pdf",
"file_path": "/uploads/certificates/2026/05/file.pdf",
"storage_provider": "local"
}
