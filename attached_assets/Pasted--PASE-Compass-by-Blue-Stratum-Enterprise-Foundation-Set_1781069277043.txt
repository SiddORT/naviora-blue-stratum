# Authorization

Role Based Access Control.

Tables:

```sql
roles
permissions
role_permissions
user_roles
```

Create permission middleware.

Example:

```text
users.view
users.create
users.edit
users.delete

organizations.view
organizations.create
```

Permissions must be dynamic and database driven.

No hardcoded permissions.

# File Storage Standards

IMPORTANT:

Never store files as blobs inside PostgreSQL.

Store only metadata.

Example:

```json
{
  "file_name": "certificate.pdf",
  "original_name": "certificate.pdf",
  "file_size": 50000,
  "mime_type": "application/pdf",
  "file_path": "/uploads/certificates/2026/01/file.pdf",
  "storage_provider": "local"
}
```

---

# Storage Architecture

Create storage abstraction.

Interface:

```python
StorageProvider
```

Methods:

```python
upload()
delete()
exists()
get_url()
```

Implement:

```python
LocalStorageProvider
```

Future:

```python
S3StorageProvider
AzureStorageProvider
MinIOStorageProvider
```

Business logic must never know which storage provider is being used.
