# Business Concept

An Assessment is a collection of one or more Exercises.

Example:

Assessment:
COLREG Level 1 Assessment

Contains:

* Basic Crossing Assessment
* Basic Head-On Assessment
* Basic Overtaking Assessment

The assessment defines:

* Pass Criteria
* Duration
* Attempts
* Variant Selection Rules
* Assessment Instructions

Assessments do NOT belong to organizations.

Assessments are platform assets created by Admin and later assigned to organizations.
